# Digital Time Study App

A web-based time study application for recording and analyzing work activities with speech-to-text support.

## Quick Start

Open the app:
```
https://konradandrag-max.github.io/time-study-auto/time-study.html
```

## Features

- **Record Activities**: Log work activities with automatic speech-to-text
- **Classify Activities**: Categorize as Internal, External, Waste, or Delay
- **Analytics**: View efficiency trends, operator comparisons, and waste analysis
- **Dark Mode**: Comfortable viewing in low-light environments
- **Data Export**: Export studies as CSV or JSON
- **Offline Data**: All data stored locally in your browser

## How It Works

1. Click **"DO TIME STUDY"** to start recording
2. Enter Date, Item Master No, and Operator
3. Click **"Start Study"** to begin
4. Speak your activities into the microphone
5. Review and classify activities
6. Save your study
7. View analytics on the dashboard

## Requirements

- Modern web browser (Chrome, Safari, Firefox, Edge)
- Internet connection (for speech-to-text)
- Microphone access (with browser permission)

## Data Privacy

✅ All data is stored locally in your browser  
✅ No data is sent to external servers  
✅ No account or login required

## Deployment

The app is hosted on GitHub Pages. To deploy your own copy:

1. Fork this repository
2. Enable GitHub Pages in Settings → Pages
3. Set source to `main` branch
4. Access your app at: `https://yourusername.github.io/time-study-auto/time-study.html`
